# Contributing

Thanks for your interest in contributing! This is an Electron + React + Vite app written in TypeScript.

## Getting Started
```bash
npm install
npm run dev
```

## Build and Package
```bash
npm run build
npm run dist
```

## Branches
- `main`: stable releases
- feature branches: `feature/<short-desc>`
- fix branches: `fix/<short-desc>`

## Pull Requests
1. Keep PRs focused and small where possible
2. Include a clear description and screenshots for UI changes
3. Ensure:
   - `npm run typecheck` passes
   - `npm run lint` passes
   - `npm run build` succeeds

## Code Style
- TypeScript strict mode
- ESLint + Prettier formatting
- Prefer meaningful names and guard clauses

## Security
Please report vulnerabilities privately per `SECURITY.md`.

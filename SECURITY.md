# Security Policy

## Supported Versions
We support the latest release from `main`.

## Reporting a Vulnerability
Please email a detailed report to the maintainers. Do not open public issues for security reports.

Include:
- Affected version/commit hash
- Reproduction steps or PoC
- Impact assessment
- Suggested remediation, if known

We will acknowledge receipt within 3 business days and provide an estimated timeline for a fix.

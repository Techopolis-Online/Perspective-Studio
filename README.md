# Perspective Studio

A simple Hello World macOS application built with SwiftUI.

## Getting Started

### Prerequisites

- **macOS 13.0** or later
- **Xcode 15.0** or later
- **Swift 5.0** or later

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Techopolis-Online/Perspective-Studio.git
   cd Perspective-Studio
   ```

2. **Open in Xcode**:
   ```bash
   open "Perspective Studio.xcodeproj"
   ```

3. **Build and run**:
   - Press `⌘R` or choose Product → Run
   - The app will launch and display "Hello, World!"

## Project Structure

```
Perspective Studio/
  Perspective_StudioApp.swift    # Main application entry point
  ContentView.swift              # Main view displaying "Hello, World!"
  Assets.xcassets/               # App assets
  Info.plist                     # App configuration
  Perspective_Studio.entitlements # App entitlements
```
